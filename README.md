# 秒记账 v1.5 预算首页版

更新内容：
- 首页改为预算进度布局
- 右上角显示月份，例如 2026年4月
- 本月已用金额、预算使用率圆环、进度条
- 进度条左下角显示剩余金额
- 进度条右下角显示预算金额，可点击编辑
- 本月收入、本月结余、记账笔数、日均支出
- 快捷分类入口
- 点击消费记录时先弹出：修改 / 删除 / 取消
- 保留日历、分类管理、关于我们、备份恢复、CSV 导出
- 首次打开不预置示例记账数据

上传 GitHub 后运行 Actions，下载 `miaojizhang-v1-5-apk`。
如果 `.github` 文件夹上传失败，请手动创建 `.github/workflows/build-apk.yml`，复制 `ACTIONS_WORKFLOW_手动复制用_build-apk.yml` 内容进去。
